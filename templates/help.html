What is Flack?

Flack allows you to create channels where different users on the site can interact by sending messages.

A simple chatroom utilizing HTML, CSS, Javascript and Socket.IO. This project was made for CS50W - Project 2

SIGN IN now to start using Flack

Usage
  • Sign in using a username
  • Create a chatroom
  • View all channels and users
  • Join any chatrooms and interact with other users
  • Send an alert message to a specific user
  • Remembers the you and the channel you were on if you accidentally closed the window/browser
